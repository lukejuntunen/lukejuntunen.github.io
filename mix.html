Mix!



2014 - Vending Machine Concept rendered as Interactive Website

Mix! is inspired by the coke vending machines that have become popular due to use in restaurants such as 5 Guys Burgers and Fries and stores such as Wegmans. The conceptual idea is to create an interface that allows for the same customization, but also allows for easy mixing of drinks into a custom order. This concept is currently a work in progress and the final version is not yet complete.
