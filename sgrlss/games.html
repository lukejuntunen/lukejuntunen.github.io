SGRLSS IS CURRENTLY IN EARLY DEVELOPMENT ON A NUMBER OF EXCITING PROJECTS INCLUDING:


immediate area walking simulator 2014



Immediate Area Walking Simulator is an upcoming arcade game scheduled for release in Spring 2014. The game seeks to allow users to gain the healthy benefits of walking through the simulation of walking in the immediate area. The game is currently in development alongside an arcade console for installation in homes and businesses, along with a possible mobile version for activity on the go.

40 winks



40 Winks is a game based on the medical principles surrounding getting a good nights sleep. The game places you in a bedroom and allows you to go to sleep and make sure you get a full 8 hours for maximum rest! Look out for this game on sgrlss arcade machines as well as an open internet beta coming soon!

focus



Focus is an eye heath and reaction game planned for release in Spring/Summer 2014. The game will feature a variety of tests and games meant to improve eyesight and train reaction times based on visual stimuli. This game is currently in pre-development and will begin early programming in the near future.
